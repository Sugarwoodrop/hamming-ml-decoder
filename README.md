# Условие
Запрограммировать декодер работающий по принципу максимума правдоподобия для кода Хэмминга с r проверками на четность, основанный на вычислении синдрома. 
Вы сами выбираете какой конкретно код Хэмминга будет использован (порядок столбцов проверочной матрицы H), но при этом не должны создаваться проверочная матрица H и таблица синдромов. Вычисление синдрома должно быть организовано динамически.

# Ввод
Текстовый файл input.txt, содержащий число проверок на четность r и вектор, который необходимо декодировать.

# Пример ввода:
3
[1,1,1,1,0,1,1]

# Вывод
Текстовый файл output.txt, содержащий вектор в прежнем формате.
В отдельной последней строке общее время выполнения программы в миллисекундах вида Time: 12345ms.

Необходимо наличие исполняемого файла формата .exe или инструкция по сборке.
Упор сделать на декодирование очень длинных кодов.

# PS
**В generator создание input файла с произвольным r и единичный вектор с ошибкой в позиции которую можно указать**
